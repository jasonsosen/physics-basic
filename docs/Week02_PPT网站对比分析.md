### 4. SI基本単位补全
网站只写了 3 个（m, kg, s），PPT 有完整的 7 个。

**需要补全：**
| 単位 | 記号 | 物理量 |
|------|------|--------|
| メートル | m | 長さ |
| キログラム | kg | 質量 |
| 秒 | s | 時間 |
| **アンペア** | **A** | **電流** ← 漏了 |
| **ケルビン** | **K** | **温度** ← 漏了 |
| **モル** | **mol** | **物質量** ← 漏了 |
| **カンデラ** | **cd** | **光度** ← 漏了 |

### 5. SI組立単位
PPT 有一页专门讲组立単位，网站完全没有。

**建议添加：**
```
力: N (ニュートン) = kg·m/s²
圧力: Pa (パスカル) = N/m²
エネルギー: J (ジュール) = N·m
```
