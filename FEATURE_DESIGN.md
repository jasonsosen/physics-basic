## 🎯 Week 02: 単位換算トレーナー

### 功能描述
**消消樂式單位換算練習器** - 拖拽單位互相約分

### 實際題目
```
① 1.0 km → m
② 1.0 m → mm  
③ 1.0×10⁻¹ μm → nm
④ 1.0×10⁴ cm² → m²
⑤ 1.0×10² g·cm/s² → N
⑥ 1.0 g/cm³ → kg/m³
```

### 交互設計
```
┌─────────────────────────────────────────┐
│  1.0 km  ×  [    ?    ]  =  [    ] m   │
│                                         │
│  可拖拽的換算因子:                       │
│  ┌──────────┐  ┌──────────┐             │
│  │ 1000 m   │  │   1 km   │             │
│  │ ─────── │  │ ─────── │             │
│  │  1 km    │  │  1000 m  │             │
│  └──────────┘  └──────────┘             │
│                                         │
│  拖入正確的因子，看單位互相消掉！        │
└─────────────────────────────────────────┘
```

### 錯誤診斷
| 學生答案 | 診斷 |
|---------|------|
| 1.0×10² m | "km比m大，換算後數字應該變大！" |
| 1.0×10⁶ m | "1 km = 1000 m，不是1000000" |

### 特殊處理: 面積/體積換算
```
cm² → m² 時顯示:
  1 cm = 0.01 m
  1 cm² = (0.01)² m² = 0.0001 m²
  
  動畫: 正方形邊長縮小，面積變化
```

## 🎯 Week 03: 密度公式三角形

### 功能描述
**公式變形訓練器** - 點擊要求的量，自動顯示變形公式

### 實際題目
```
① 體積40cm³，質量200g → 密度?
② 密度4.0g/cm³，質量200g → 體積?
③ 密度3.0g/cm³，質量300g → 體積?
④ 密度3.0g/cm³，體積20cm³ → 質量?
```

### 交互設計
```
        ┌───┐
        │ m │  ← 點擊這裡求質量
        └───┘
       ╱     ╲
      ╱       ╲
   ┌───┐    ┌───┐
   │ ρ │    │ V │  ← 點擊求體積
   └───┘    └───┘

點擊後顯示:
  求 V → V = m/ρ
  帶入: V = 200g ÷ 4.0g/cm³ = 50 cm³
```

### 陷阱題: 質量vs重量
```
問: 月面上物體A的質量是多少kg?

❌ 常見錯誤: 0.48 kg ÷ 6 = 0.08 kg
✅ 正確答案: 0.48 kg (質量不變!)

彈出動畫:
  地球上的你: 60 kg (秤顯示60)
  月球上的你: 60 kg (秤顯示10)
  
  "質量是你身上有多少肉，去哪都不變！"
  "重量是星球拉你的力，會變！"
```

## 🎯 Week 04: 向量合成動畫

### 功能描述
**向量加法視覺化** - 平行四邊形法則 + 成分計算

### 實際題目
```
F₁ = (5.0, 0.0) N
F₂ = (-2.0, 4.0) N
求: 合力 F 和 |F|
```

### 交互設計
```
     y
     ↑     F₂
     │    ↗
     │   /
     │  /
     │ /
─────┼─────→ x
     │  F₁→
     │
     
步驟1: 顯示F₁ (紅色箭頭)
步驟2: 顯示F₂ (藍色箭頭)  
步驟3: 平移F₂到F₁終點
步驟4: 顯示合力F (綠色箭頭)
步驟5: 計算 |F| = √(3² + 4²) = 5
```

### 力的分解
```
|F| = 10N, θ = 120° (與x軸)

動畫顯示:
  Fx = |F|cos(120°) = 10 × (-0.5) = -5 N
  Fy = |F|cos(30°) = 10 × (√3/2) ≈ 8.66 N
  
滑桿: 拖動角度，即時更新分量
```

## 🎯 Week 06: 張力平衡器

### 功能描述
**受力分析 + 平衡方程式**

### 實際題目
```
球重3.0N，水平拉力4.0N
求: 傾斜時的張力
```

### 交互設計
```
        ╲ T
         ╲
          ╲
           ●  ← 3.0N (重力)
          →
         4.0N (水平拉力)

步驟引導:
  1. 畫出所有力 ✓
  2. 列平衡方程式:
     水平: Tx = 4.0 N  [填空]
     鉛直: Ty = 3.0 N  [填空]
  3. 求張力大小:
     T = √(Tx² + Ty²) = √(16+9) = [5.0] N
```

### 錯誤診斷
| 學生答案 | 診斷 |
|---------|------|
| T = 7.0 N | "張力的兩個分量不能直接相加！用勾股定理" |
| T = 1.0 N | "張力要平衡兩個力，應該比單個力更大" |

## 🎯 Week 09: 圖像讀取訓練

### 功能描述
**x-t圖 / v-t圖 互動式讀取**

### 實際題目
```
從x-t圖讀取:
① t=0~10s 的速度
② t=10~15s 的速度
③ t=0~15s 的平均速度
```

### 交互設計
```
x(m)
  │
30├───────●
  │      /│
20├─────/ │
  │    /  │
10├───/   │
  │  /    │
 0├─●─────┼─────→ t(s)
  0  10  15  20

點擊兩點，自動計算斜率:
  v = Δx/Δt = (30-0)/(10-0) = 3.0 m/s
  
高亮顯示: "斜率 = 速度"
```

### v-t圖特殊功能
```
v(m/s)
  │
 8├───●───●
  │  /     ╲
 4├─/       ╲
  │/         ╲
 0├───────────●──→ t(s)
  0   4   8   12

面積著色:
  "陰影面積 = 位移"
  
點擊區間，自動計算:
  面積 = (1/2)(4)(8) + (8)(4) = 48 m
```

## 🎯 Week 10: F=ma 計算器

### 功能描述
**運動方程式三變形**

### 實際題目
```
① m=2kg, a=3m/s² → F=?
② m=2kg, F=-4N → a=?
③ a=1m/s², F=3N → m=?
```

### 交互設計
```
┌─────────────────────────────┐
│     F  =  m  ×  a           │
│    [ ]   [2]   [3]          │
│                             │
│  已知: m, a                  │
│  求: F                       │
│                             │
│  [計算] → F = 2 × 3 = 6 N   │
└─────────────────────────────┘

三個輸入框，填任意兩個，自動求第三個
```

## 🎯 Week 12: 鉛直運動模擬器

### 功能描述
**投上/自由落體 動畫 + 公式選擇**

### 實際題目
```
v₀=10m/s 投上, g=10m/s²
① 最高點時間
② 最高點高度  
③ 落地時間
④ 落地速度
```

### 交互設計
```
        ↑ v₀=10m/s
        │
        ●  ← 點擊開始
        │
════════╧════════ 地面

動畫播放:
  t=0: v=10 ↑
  t=0.5: v=5 ↑
  t=1.0: v=0 (最高點) ★
  t=1.5: v=5 ↓
  t=2.0: v=10 ↓ (落地)
  
公式選擇器:
  求最高點時間 → 選 v=v₀-gt，令v=0
  求最高點高度 → 選 h=v₀t-½gt²
```

### 對稱性視覺化
```
上升時間 = 下降時間
初速度 = 落地速度

動畫: 軌跡左右對稱
```

## 🎯 綜合功能: 步驟引導解題器

### 設計原則
每道題強制分5步:

```
Step 1: 讀題提取信息
  "劃出已知量" → 學生點擊文字高亮
  
Step 2: 畫圖建模
  "畫受力圖" → 拖拽箭頭
  
Step 3: 選公式
  "從公式庫選擇" → 點選正確公式
  
Step 4: 代入計算
  "先寫字母式，再代數字"
  
Step 5: 檢查答案
  "你的答案合理嗎？"
  (博爾特跑300m/s → 警告)
```
